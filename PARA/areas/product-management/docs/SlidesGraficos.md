\# Mardraw Consolidado Modelo ADKAR



\## Esquema Jerárquico

\[Inicio] → \[ADKAR]

&#x20;             ├─ Awareness (Conciencia)

&#x20;             │      ├─ Reconocer la necesidad del cambio

&#x20;             │      ├─ Identificar riesgos si no se actúa

&#x20;             │      └─ Comunicar claramente el motivo

&#x20;             │

&#x20;             ├─ Desire (Deseo)

&#x20;             │      ├─ Motivar la participación

&#x20;             │      ├─ Generar compromiso personal

&#x20;             │      └─ Mostrar beneficios individuales y colectivos

&#x20;             │

&#x20;             ├─ Knowledge (Conocimiento)

&#x20;             │      ├─ Capacitación en nuevas habilidades

&#x20;             │      ├─ Documentación y guías de apoyo

&#x20;             │      └─ Transferencia de buenas prácticas

&#x20;             │

&#x20;             ├─ Ability (Habilidad)

&#x20;             │      ├─ Aplicar lo aprendido en la práctica

&#x20;             │      ├─ Superar barreras iniciales

&#x20;             │      └─ Acompañamiento y coaching

&#x20;             │

&#x20;             └─ Reinforcement (Refuerzo)

&#x20;                    ├─ Reconocer logros

&#x20;                    ├─ Establecer métricas de seguimiento

&#x20;                    ├─ Recompensas y retroalimentación

&#x20;                    └─ Mantener la sostenibilidad del cambio
